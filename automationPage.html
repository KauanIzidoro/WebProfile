Arduino Access Control

This project was developed using the Arduino microcontroller ecosystem for control, an LCD display as the human-machine interface (HMI), and a matrix keypad as a peripheral. The objective is to implement an access control system with four menu options. The system should include an administrator user with privileges superior to those of other users.

ESP Clock and Counter

This project was developed using the ESP-32 microcontroller and an LCD display to meet the following requirements: Create a program that includes three menu options as described in the video. The user can choose between counting clicks, displaying a clock, or clearing the data.
