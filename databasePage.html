Database

Educational System

In this project, we were asked to implement the modeling of a database for a university educational system. After designing the diagram, the implementation of DDL, DML, and DQL codes becomes simpler. This project included relationships with NxN and 1xN cardinality.

NoSQL queries

In this project, the use of NoSQL databases was put into practice, specifically MongoDB, through Compass GUI and MongoSH. This allowed us to gain knowledge of how a NoSQL database functions and how to perform queries following the JSON document model.
