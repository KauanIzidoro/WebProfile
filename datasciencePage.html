Data Science

Python Google Colab

All data science projects were conducted using the Google Colab environment, which is cloud-based and provides access to advanced hardware such as GPUs and TPUs for project applications. Among the tasks, we were introduced to key analysis techniques as well as the fundamentals of data science and statistics.

Pandas lib

Pandas is an open-source software library written for the Python programming language for data manipulation and analysis. It offers data structures and operations to manipulate numerical tables and time series. It is widely used in data science, data analysis, and machine learning, thanks to its ability to handle large volumes of data efficiently. Our interest was in its functions and data structures such as DataFrames.
