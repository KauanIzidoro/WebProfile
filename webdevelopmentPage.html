Document Object Model

The Document Object Model (DOM) is a programming interface that allows scripts to access and update the content, structure, and style of HTML and XML documents. It represents the document as a tree of nodes, where each node corresponds to a part of the document (such as elements, attributes, and text).

Iframe

An (inline frame) is an HTML element that allows embedding another HTML document within a web page. It functions as a rectangular window on a web page, displaying the content of another document independently of the rest of the page. The content of the is defined by the src attribute, which specifies the URL of the document to be embedded.
